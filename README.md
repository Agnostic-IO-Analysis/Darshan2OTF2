# darshan-to-otf2

## Dependencies
* python 3.9.4
* pydarshan
* otf2

## Usage

```shell
python3 darshan_to_otf2 <args> <darshan_trace>
```

## Args
|short option | long option | explanation | 
| --- | --- | --- |
| -o | --output | specifies output location (default is ./trace_out) |
| -t | --timer | sets timer resolution (default is 1e9) |
